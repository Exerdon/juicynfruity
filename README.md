# Juicy N Fruity – Interactive baverage Brand Website

## Overview
Juicy N Fruity is a responsive and interactive website developed using HTML, CSS, and JavaScript. The project focuses on delivering an engaging user experience with modern UI design, smooth interactions, and visually appealing layouts for a fruit-based brand/business concept.

## Features
- Responsive design for desktop and mobile
- Interactive UI components
- Smooth navigation
- Modern landing page design
- Optimized visual layout

## Problem It Solves
The project demonstrates how a visually engaging website can improve digital branding and user experience for a food/product-based business.

## Tech Stack
- HTML5
- CSS3
- JavaScript
